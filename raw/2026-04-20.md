# 记忆进化任务

## 任务目标

OpenClaw-UltraMemory V4 的核心目标：让记忆系统具备自我进化能力，参考 Hermes Agent 的自动技能生长机制。

## 核心需求

1. **热度钉扎滑动窗**：1000 条上限，7 天保护期，自然衰减
2. **自动技能生成**：同类问题解决 3 次 → 自动写 SKILL.md
3. **子智能体并行**：多步骤任务自动拆分并发执行
4. **防迷失锚点**：instruction_hash 检测偏离自动拉回
5. **不可篡改**：raw/ 只追加，DB 只改索引不动原文
6. **可迁移**：export.sh / import.sh / verify.sh 一键迁移

## 演进路径

- V1：SimHash 检索
- V2：Parent_ID 链路回溯 + instruction_hash 锚点
- V3：热度滑动窗（当前 PROTECT_SECS=604800）
- V4：融合 Hermes 自动技能生长 + 子Agent并行

## 铁律

1. 原始数据不可篡改
2. 行为全程可审计

## 生态定位

唯一不可篡改 + 自动进化 + 完整可迁移的记忆系统，区别于官方和其他 Claw 变种的核心壁垒。
